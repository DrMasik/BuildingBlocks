Identify Users who have had recent changes to their assigned records.

Summary:

This endpoint identifies Users who have had recent changes to their assigned records. Returns an array of User IDs based upon the filter criteria.

Request Arguments

Name Type Description Required
since Timestamp (String) Selects users with records modified since this time. True
module String A comma separated list of modules that will be checked. Modules must be assignable and therefore contain the assigned_to_user field. True
include_deleted Boolean Determines if recently deleted records should be included as part of recent changes. True

Request

http://{site_url}/rest/Users/recentChanges?since=2018-02-14T12:00:00Z&modules=Meetings,Calls&include_deleted=1
Note: GET endpoint parameters are passed in the form of a query string.

Response Arguments

Name Type Description
<User IDs> Array<String> An array of User ID strings.
